# PontoCertoBeta-atv4UC15
Parte do projeto com conexão a banco de dados MySQL - para entrega de atividade

PONTO CERTO

Status do projeto (Em desenvolvimento, mas funcional

Devs : Maicon C. Z.

O sistema proposto é um aplicativo desktop de controle de ponto para funcionários de uma empresa. Seu objetivo principal é registrar os horários de entrada e saída dos colaboradores, armazenando essas informações em um banco de dados. O sistema permitirá o cadastro de funcionários, registro de horários (ponto) e gestão de dados trabalhistas como salários e carga horária. O acesso ao sistema será feito por meio de login, com dois níveis de permissão: administrador e funcionário.

FUNÇÕES
1.	Cadastro e autenticação de usuários (admin e funcionários)
2.	Registro de ponto (entrada e saída)
3.	Consulta e gerenciamento de dados dos funcionários (salário, jornada de trabalho, etc.)

Descrição dos Usuários
•	Administrador: Acesso total ao sistema, podendo cadastrar/editar/deletar funcionários, consultar registros de ponto, definir salários e horários.
•	Funcionário: Pode acessar o sistema apenas para registrar ponto e visualizar seus próprios dados e histórico de ponto.

Requisitos Funcionais
•	Cadastro de funcionários com login, senha, salário e jornada de trabalho.
•	Login com autenticação de dois níveis (admin ou funcionário).
•	Registro de ponto (entrada/saída) com registro de data e hora.
•	Consulta de histórico de ponto (individual e geral).

Programas Uilizados
•	Linguagem de programação: Java
•	Banco de dados: MySQL
•	Interface: Desktop (NetBeans)
